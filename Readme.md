## cocos creator 小游戏

参考了各类的小游戏源码和demo,并尝试着用自己设计方法实现完成

大部分都是学习时做的,可能注释做的不太好,代码会慢慢整理的

主要用于学习,使用的代码都不多

引擎版本 2.4.3 -- 2.4.5
